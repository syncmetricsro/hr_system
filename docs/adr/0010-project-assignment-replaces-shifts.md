# ADR 0010: Project Assignment Replaces Shifts

Status: Accepted

## Context

Daily shifts, second shifts, vehicles, and bus rosters are removed from production scope.

## Decision

Project assignment is the operational assignment unit. One worker has one active project at a time, with history retained later.

## Consequences

No shift, fleet, or capacity models are added. Later transport work records weekly headcounts, not passenger rosters.
